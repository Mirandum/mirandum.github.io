Nuestro proyecto

El proyecto que realizaremos para la clase ingeniería de software se trata de una tienda de accesorios online; con accesorios nos referimos a relojes, pulseras, aretes, gafas y collares.

El usuario contará con un carrito de compras y unicamente podrá comprar si esta registrado en la tienda. Para la modalidad de tienda usaremos el enfoque e-commerce.

La pagina web tiene que ser responsive, es decir que se pueda ver en cualquier dispositivo sin que se pierda información por el tamaño de la pantalla.

Por el lado del front-end, es decir la parte que interactua con el usuario se usaran herramientas como Bootstrap y Font-awesome.

Por el lado del back-end, es decir donde se hace toda la administración de los datos y las peticiones usaremos un servidor llamado infinity y para las bases de datos Sql y phpMyadmin.
